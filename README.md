# Animalerie (site + admin + Stripe)

## Installation locale
```bash
npm install
npm start
```
Puis ouvrir : http://localhost:3000/index.html

## Admin
- Page : /admin.html
- Mot de passe démo : admin123
- Les produits sont stockés dans data/produits.json

## Images produits
- Dépose tes images dans le dossier images/
- Dans l'admin, renseigne : images/nom_image.jpg

## Stripe (paiement)
- Variables d'env : STRIPE_SECRET_KEY, DOMAIN (optionnel)
